# Cowrie-Honeypot
A Cowrie Honeypot that enables SSH connection with attacker so that we can monitor their attack patterns and behavior. This will help us in performing behavior analysis.

# Honeypot Deployment Project
## Overview

Deployed and monitored a honeypot environment during internship to observe malicious activities and unauthorized access attempts.

## Tools Used
- Raspberry Pi
- Linux
- Cowrie 
- Logging tools

## Responsibilities
- Honeypot setup
- Monitoring incoming connections
- Reviewing attack logs
- Analyzing suspicious activities
